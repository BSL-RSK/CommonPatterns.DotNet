# 🙋 Contributing

If you're adding a new pattern, test template, or utility:

1. Use an appropriate subfolder in `src/`, `tests/`, or `docs/`
2. Include a local `README.md` explaining its purpose and usage
3. Follow naming conventions used across the repo
4. Make a pull request from a feature branch into `develop`
5. Self-review or request one if needed

Thank you for improving our shared patterns ✨
